To Run SPLint for the first time, SPLint environment variables must be added first

1.
Variable name: LARCH_PATH
Variable value: <directory where you installed splint>\<PhilRobkit Checkouts>\trunk\externals\tools\splint\lib


2.
Variable name: LCLIMPORTDIR
Variable value: <directory where you installed splint>\<PhilRobkit Checkouts>\trunk\externals\tools\splint\imports



To Run SPLint simply invoke the "CodeCheck PhilRobokit.bat" batch file...
